## Registrarium — Access notice

You have been granted access to the Archivist’s computer.

This terminal holds the logs, maps, and memory fragments gathered before its owner disappeared. It has not been altered. Gaps, missing entries, and inaccessible districts are documented as found.

Navigation is simple. It was meant to be used alone, in haste or in the dark.

---

## Navigation

| Screen            | What you see                         | How to proceed                                                         |
| ----------------- | ------------------------------------ | ---------------------------------------------------------------------- |
| Desktop           | Main registrarium menu               | Click a file                                                           |
| District map      | Aerial view of the vertical city     | Click a district — or use arrow keys + Enter                           |
| District          | Interactive scene                  | Click active zones                                                     |
| Story             | Scrolling text                     | Scroll wheel or drag — back button top left                            |

---

## Notes

- The camera, when available, enables parallax. Otherwise: device orientation or mouse.
- Some districts are empty. That is not a bug.
- The archives are numbered. They are not sorted by importance.

---

## About the author

The Archivist began this journal when reading, gathering, or any leisure activity was forbidden by T-CORP Corp. They documented not to be read, but so forgetting would not be total.

> “If the recovered manuscript is a well-worn trope in literature, what of the recovered computer?”
> — LOG 02
